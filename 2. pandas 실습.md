#### 4>  결측치 처리법

- 결측치 처리

  >  null data == 빠진 데이터  
  > -> 결측치 처리 (Missing data handling)
  >
  > - 랜덤 값으로 채우기 
  > - mean/median 값으로 채우기 (age, salary, ...)
  > - 구간별로 나눠서 구간의 대표값(mean/median)으로 : 직원들 데이터 -> 직급 열의 구간 
  > - ML/DL 모델을 활용해서 예측해서 넣기 : age열에 결측치 존재 -> age열 제외 나머지 열로 예측
  > - 결측치가 50~70% 이상인 열의 경우 : 삭제도 고려
